## 🚀 Common Workflows

### Workflow 1: Hiring a New Employee (5-10 minutes)

**Goal:** Move an applicant from "Shortlisted" to "Employee Activated" + directory

#### Step 1: Find the Applicant
1. Click **"Applicants"** in navigation
2. Look for applicant in list
3. Click applicant card to open details

#### Step 2: Schedule Interview (optional but recommended)
1. In applicant details, click **"Schedule Interview"**
2. Enter: Date, Time, Interviewer name
3. Click **"Schedule"**
4. ✅ Interview scheduled

#### Step 3: Move to "Selected" Status
1. Change status dropdown to **"Selected"**
2. Click Save
3. ✅ Applicant now ready for offer

#### Step 4: Send Job Offer
1. Click **"Send Job Offer"** button
2. Fill in offer details:
   - Position: (auto-filled)
   - Department: (auto-filled)
   - Start Date: *Pick a date 2-4 weeks out*
   - Salary Offer: *Your offer amount*
   - Contract Type: Full-Time/Part-Time
   - Benefits: (select as applicable)
3. Click **"Create and Send Offer"**
4. ✅ Offer sent, status changes to "Offer Sent"

#### Step 5: Accept Offer
1. If applicant accepts (via email):
   - Change status to **"Offer Accepted"**
   - OR click **"Accept Offer"** button (if available)
2. ✅ Ready for hiring

#### Step 6: Create Employee Account (CRITICAL STEP)
1. Click **"Hire & Create Employee Account"** button
2. **Wait 3-5 seconds** for processing
3. ✅ See success message with:
   - Employee ID (e.g., EMP-2026-0001)
   - Username (for login)
   - Temporary Password
4. **⚠️ IMPORTANT:** Save/share credentials with new hire
5. ✅ Status changes to "Hired"

#### Step 7: Verify Onboarding Started
1. Click **"Onboarding"** in navigation
2. Find new employee under **"Offer Accepted"** section
3. Verify 6 tasks listed:
   - ☐ Submit Government IDs
   - ☐ Upload Medical License
   - ☐ Sign Employment Contract
   - ☐ Complete HR Orientation
   - ☐ Receive Employee ID Badge
   - ☐ Payroll Registration
4. ✅ Onboarding started automatically

#### Step 8: Complete Onboarding
1. As tasks completed, check them off
2. When last (6th) task checked:
   - Employee auto-promoted to "Employee Activated"
   - Disappears from Onboarding page
3. ✅ Employee now in "Employees" directory

**Total time:** ~5-10 min active work (hiring)
**Passive time:** Days-weeks (onboarding tasks)

## 📅 Onboarding Page

### Pipeline Summary (Top)

Shows employee count at each stage:

| Stage | Meaning | Your Action |
|-------|---------|-------------|
| **Offer Accepted** | Just hired | Send credentials, kickoff |
| **Documents Submitted** | Uploading docs | Verify docs received |
| **Orientation Completed** | Training done | Sign off |
| **Employee Activated** | Ready to work | Add to directory |

### Employee Cards

Each card shows:
- Avatar with initials
- Name
- Position & Department
- Progress bar (e.g., "3 of 6 Complete")
- Expandable sections

### Expand Card to See Sections

**1. Onboarding Progress**
- Visual 4-stage pipeline
- Green checkmarks show completion
- Percentage complete

**2. Onboarding Checklist**
- List of 6 tasks
- Check boxes to mark complete
- As you check: Progress bar updates
- When all checked: ✅ Auto-promoted!

**3. Document Upload**
- Upload government IDs
- Upload medical licenses
- Upload contracts
- Upload other docs

**4. Orientation Section**
- Schedule orientation session
- Set date, time, location
- Assign trainer
- Mark completed

### Mark Tasks Complete

1. Expand "Onboarding Checklist" tab
2. Check box next to task
3. Progress bar updates immediately
4. ✅ When all 6 checked → Employee auto-activated!

**That's it!** Employee automatically:
- Updates status to "Employee Activated"
- Moves to Employees directory
- No manual promotion needed!

## ⚠️ Important Rules

### Hiring Pipeline
- ✅ Must move through stages in order
- ❌ Can't skip Explorer Interview Scheduling
- ✅ Can jump back to previous stage if needed
- ❌ Can't hire without "Offer Accepted" status

### Onboarding Tasks
- ✅ Must have 6 tasks (auto-created)
- ✅ Marked complete = task done
- ⚠️ When all 6 complete → employee auto-activated
- ✅ Auto-activation is automatic (no manual step)

### Employee Directory
- ✅ Only shows "Employee Activated" status
- ✅ Automatically updated when onboarding complete
- ❌ Won't show in training or before activated
- ✅ Can search/filter all activated employees

### Account Creation
- ✅ User gets real email (same as applicant)
- ✅ Temporary password provided (share with new hire)
- ✅ Employee can log in after password reset
- ✅ System creates both auth user AND employee record

## 🆘 Troubleshooting

### Problem: "Hire" button won't appear

**Check:**
1. Is applicant status exactly "Offer Accepted"?
   - If no: Click "Accept Offer" button
   - If yes: Continue to #2
2. Does applicant have a job offer?
   - If no: Send job offer first
   - If yes: Continue to #3
3. Refresh page (F5) and try again

**Solution:** Status must be "Offer Accepted" + offer must exist

---

### Problem: Employee doesn't appear in directory after hiring

**Check:**
1. Go to Onboarding page
2. Is employee listed under "Offer Accepted"?
   - If yes: Employee still onboarding (expected)
   - If no: Continue to #3
3. Mark all 6 tasks complete
4. Refresh page - employee now in directory

**Solution:** Employee appears in directory only after onboarding complete

---

### Problem: "Job offer not found" error when hiring

**Check:**
1. Go back to applicant
2. Did you create a job offer?
   - If no: Click "Send Job Offer" button
   - If yes: Continue to #3
3. Is offer status "Offer Sent" or "Offer Accepted"?
   - If rejected: Create new offer
   - If accepted: Try again

**Solution:** Must create offer first, then accept it

---

### Problem: Can't see Onboarding page

**Check:**
1. Are you logged in as HR/Admin?
   - If no: Log in with HR account
   - If yes: Continue to #2
2. Do you have any employees onboarding?
   - If no: Hire someone first (Test Tests above)
   - If yes: Try refresh (F5)

**Solution:** Needs HR/Admin role and employee data

---

### Problem: Employee credentials lost

**Note:** Credentials shown once after hire
- Employee ID: Check the success toast or Onboarding page
- Username: Can be reset by admin in Auth settings
- Password: Must be reset by employee via "Forgot Password" link

**Solution:** Employee can use "Forgot Password" to reset

## 🎯 Tips & Best Practices

### Tip 1: Batch Processing
Group similar tasks:
- Interview all candidates Monday
- Send offers Tuesday  
- Follow up Thursday
- Hire + onboard Friday

### Tip 2: Use Comments
Add notes in applicant dialog:
- Interview feedback
- Salary negotiation details
- Start date confirmations
- Help future reference

### Tip 3: Document Uploads
Use Onboarding → Document Upload:
- Keeps all docs in one place
- Organized by employee
- Searchable history
- Compliance-ready

### Tip 4: Monitor Pipeline
Check Dashboard regularly:
- See stuck applications
- Track time-to-hire
- Identify bottlenecks
- Plan hiring pace

### Tip 5: Weekly Checklist
- Monday: Review new applicants
- Tuesday: Schedule interviews
- Wednesday: Interview & evaluate
- Thursday: Send offers
- Friday: Follow up on acceptances
